# **App Name**: Orbital Gallery

## Core Features:

- Interactive 3D Menu: Display a rotating 3D sphere with images as interactive menu items.
- Item Linking: Link each image to a URL that opens in a new tab when selected.
- Dynamic Item Highlighting: Highlight the currently selected item as the user interacts with the rotating sphere.

## Style Guidelines:

- Background color: Dark gray (#222222) to provide a neutral backdrop and contrast with the interactive elements.
- Primary color: Bright cyan (#00FFFF) for the button.
- Accent color: Electric purple (#BF00FF) as a contrast to the bright cyan.
- Body and headline font: 'Inter', a sans-serif font for clear readability and a modern feel.
- Note: currently only Google Fonts are supported.
- Simple, geometric icons for button interactions.
- Full-screen canvas with centrally located interactive sphere.
- Smooth, subtle rotation animations to draw user attention and add dynamism.
